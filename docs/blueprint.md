# **App Name**: MITTY Virtual Studio

## Core Features:

- Product Configuration Panel: Configure product details using intuitive selection menus (Product Category, Sleeve Type, Gender, Fabric Type) and image upload.
- AI Image Generation: Generate three photorealistic images (Front, Side, Back views) using a consistent model and the uploaded product image. Also generate a HD flat lay of the uploaded image using a tool and enhance image quality with an upscaler.
- Auto-Generate Product Details: Automatically generate a product title starting with "Mitty" and a descriptive product description, including color, fabric, pattern, and fit details, leveraging a tool.
- Automated ZIP Packaging: Automatically package all generated images and product information into a ZIP file named after the product title for easy download and use.
- Image Regeneration & Individual Downloads: Enable the ability to regenerate individual images if needed and download them separately for flexibility and error correction.

## Style Guidelines:

- Primary color: A muted, sophisticated blue (#6A86A4) to evoke trust and professionalism.
- Background color: Light gray (#F0F2F5) to ensure a clean and neutral backdrop.
- Accent color: Soft gold (#C4A77D) to highlight key interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, and highly readable interface. Note: currently only Google Fonts are supported.
- Drag-and-drop interface with a minimalistic design to ensure a fast and intuitive user experience.
- Use a set of simple, clear icons that align with MITTY's brand, focusing on usability and visual appeal.
